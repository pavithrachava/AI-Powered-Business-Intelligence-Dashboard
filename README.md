# AI-Powered-Business-Intelligence-Dashboard
End-to-End Sales Analytics Dashboard using SQL, Tableau, and Business Intelligence Reporting

Project Overview-
This project analyzes retail sales data using SQL and Tableau to generate business insights related to sales performance, profitability, customer behavior, product trends, regional performance, and discount effectiveness.
